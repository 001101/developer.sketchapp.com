This class represents Sketch, and provides access to:

  • the document model and the layer tree
  • metadata abound sketch itself
  • utilities for interacting with the user
  • access to the running plugin, it’s resources and settings

Returns the version of this API.

Returns the context that the API was created in. This is the traditional context argument that is passed to all plugin scripts when they are run.

In general you should use the API to access Sketch rather than interacting with the context directly.

    get context() {
      return this.object
    }

Returns the version of Sketch that is running, eg “3.6”.

    get version() {
      return this.metadata['appVersion']
    }

Returns the exact build of Sketch that is running, eg “15352”.

    get build() {
      return this.metadata['build']
    }

Returns the full version of Sketch that is running, eg “3.6 (15352)”.

    get full_version() {
      return this.version + " (" + this.build + ")"
    }

Return the value of a global setting for a given key. @param key The setting to look up.

This is equivalent to reading a setting for the currently running version of Sketch using the defaults command line tool, eg: defaults read com.bohemiancoding.sketch3

    input(msg, initial) {

    assert(condition) {

    get selectedDocument() {
      return new Document(this.object.document, this)
    }


    newDocument() {
      var app = NSDocumentController.sharedDocumentController()
      app.newDocument_(this)
      return new Document(app.currentDocument(), this)

    }

    select(msg, items, selectedItemIndex) {
      selectedItemIndex = selectedItemIndex || 0

      var accessory = NSComboBox.alloc().initWithFrame(NSMakeRect(0,0,200,25))
      accessory.addItemsWithObjectValues(items)
      accessory.selectItemAtIndex(selectedItemIndex)

      var alert = NSAlert.alloc().init()
      alert.setMessageText(msg)
      alert.addButtonWithTitle('OK')
      alert.addButtonWithTitle('Cancel')
      alert.setAccessoryView(accessory)

      var responseCode = alert.runModal()
      var sel = accessory.indexOfSelectedItem()

      return [responseCode, sel]
    }

    message(msg) {
      this.object.document.showMessage(msg)
    }

    alert(title, msg) {
      var app = NSApplication.sharedApplication()
      app.displayDialog_withTitle(title, msg)
    }

    rectangle(x, y, w, h) {
      return new Rectangle(x, y, w, h)
    }
